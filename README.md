# GearAdvisor

GearAdvisor is a World of Warcraft addon that calculates gear upgrades directly in item tooltips.

Instead of static BiS lists or simple stat weights, GearAdvisor dynamically evaluates items based on:

• Your class and spec
• Equipped gear
• Stat caps (hit, expertise, defense)
• Slot comparisons (rings, trinkets, etc)

Example tooltip output:

Feral DPS: +24.4% slot / +5.2% overall (Ring 2)
Feral Tank: +32.4% slot / +14% overall (Ring 1)

## Features

- Shows upgrade % for every spec of your class
- Handles rings and trinkets correctly
- Stat cap scaling (hit / expertise / defense)
- Works directly in the item tooltip
- Lightweight and fast

## Installation

1. Download the latest release
2. Extract into:

Interface/AddOns/GearAdvisor

3. Restart the game
